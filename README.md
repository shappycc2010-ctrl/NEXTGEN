This archive contains a colorful starter for NEXTGEN (server + web).
## Quick run (local)
- Server:
  cd server
  npm install
  cp .env.sample .env
  fill .env with keys and APP_API_KEY
  npm run dev
- Web:
  cd web
  npm install
  cp .env.sample .env
  set REACT_APP_API_URL and REACT_APP_API_KEY
  npm start
